How I Built My Portfolio Website

August 28, 2025

In this post, I’ll share how I designed, developed, and deployed my personal portfolio and blog site using just HTML, CSS, and a bit of JavaScript.

1. Planning & Design

I started by sketching out the sections I wanted: About, Projects, Blog, Essays, and Contact. Inspiration came from other developer portfolios and minimalist design trends.

2. Coding

I structured the site using semantic HTML and styled it with modern CSS, focusing on responsiveness and accessibility. I kept the navigation simple for clarity.

3. Deployment

I chose GitHub Pages for free and easy hosting. After pushing my code to a public repository, I enabled GitHub Pages in the repo settings, and my site was live!
